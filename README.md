# starter-express-ts

Starter for [Express](https://expressjs.com/) + TypeScript

## Install

```shell
$ mkdir ${PJ_NAME}
$ cp ${PJ_NAME}
$ curl -SL https://github.com/memory-lovers/starter-express-ts/archive/master.zip | tar -xf - --strip-components 1
$ ls
# => starter-node-ts/
```
